E-Commerce Management System (Java GUI Project)

This project is a Java-based E-Commerce Management System built with a Graphical User Interface (GUI) using Swing and AWT. It allows users to browse products, manage a shopping cart, and process orders in an interactive desktop environment. The system also provides admin features to manage inventory, product details, and customer records.

Features

User-friendly GUI interface built using Java Swing

Product catalog display and category-wise filtering

Add, remove, and update items in the shopping cart

Secure user authentication (login/signup)

Admin dashboard for product and order management

Order summary and billing system

Tech Stack

Language: Java

GUI Framework: Swing / AWT

Database: MySQL (or file-based storage)
